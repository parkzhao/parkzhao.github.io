帕克胖夜话Docker-安装Docker Engine

Posted on 2016-07-28 Edited on 2019-07-04 In Docker , Docker Engine , talk Comments:

准备

安装系统必要软件

添加安装docker engine的镜像源

查看操作系统版本

执行命令 lsb_release -a

创建镜像源文件

创建docker.list文件，文件路径 /etc/apt/sources.list.d/docker.list,根据操作系统的版本，来选择相关的镜像源头

Docker安装

  • 更新 apt 包索引
    sudo apt-get update
  • 安装 docker
    sudo apt-get install docker-engine
  • 安装docker指定版本
